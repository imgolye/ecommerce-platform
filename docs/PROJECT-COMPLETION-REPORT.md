### API分布
**用户模块（6个）：**
1. POST /api/user/register - 用户注册
2. POST /api/user/login - 用户登录
3. GET /api/user/{id} - 用户信息查询
4. PUT /api/user/{id} - 用户信息更新
5. DELETE /api/user/{id} - 用户删除
6. GET /api/user/list - 用户列表

**商家模块（8个）：**
1. POST /api/merchant/register - 商家注册
2. POST /api/merchant/login - 商家登录
3. GET /api/merchant/{id} - 商家信息查询
4. PUT /api/merchant/{id} - 商家信息更新
5. POST /api/merchant/store - 店铺创建
6. GET /api/merchant/store/{id} - 店铺信息
7. PUT /api/merchant/store/{id} - 店铺更新
8. POST /api/merchant/qualification - 资质提交

**商品模块（46个）：**
- 分类管理（5个）：创建、查询树、详情、更新、删除
- 品牌管理（5个）：创建、列表、详情、更新、删除
- 商品管理（6个）：创建、列表、详情、更新、删除、状态
- SKU管理（6个）：创建、查询、详情、更新、删除、库存
- 库存管理（3个）：扣减、增加、预警
- 商品推荐（3个）：热门、新品、个性化
- 商品评价（3个）：创建、列表、统计
- 商品图片（3个）：上传、列表、删除
- 商品标签（3个）：创建、列表、删除
- 商品搜索（2个）：搜索、同步
- 商品收藏（4个）：添加、取消、列表、检查

**订单模块（11个）：**
- 订单管理（6个）：创建、详情、用户列表、商家列表、状态更新、取消
- 购物车（5个）：添加、更新数量、删除、列表、清空
